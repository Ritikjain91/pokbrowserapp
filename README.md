# Pok-mon-Browser-App
Shows a paginated list of Pokémon (using PokéAPI) • Lets the user click a Pokémon to view its details (name, id, weight, sprite) Requirements 1. Use any framework or plain JavaScript — keep it simple. 2. The list should show Pokémon names and a "View Details" button or link. 3. The details can appear inline or on a separate page.
